## Useage

#### Install

```shell
  $ yarn install
```

#### Quick Start

##### Start All

```shell
  $ yarn start
```

##### Start Part

```shell
  $ npm run start --module=user
  $ npm run start --module=user,business,common
```

### build

```shell
  $ yarn dev
  $ yarn test
  $ yarn uat
  $ yarn build
```
